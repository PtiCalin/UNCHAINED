# unchained-core

Shared platform runtime for UNCHAINED.

## Sections

- `library/` — media metadata, indexing, provenance, and storage contracts.
- `jobs/` — background queue, job lifecycle, retries, and streaming logs.
- `models/` — local model registry and orchestration surfaces.
- `plugins/` — plugin loading, capability contracts, and manifests.
- `desktop-shell/` — desktop host integration and shell-level services.

> Initial scaffold for the expanded multi-module scope.
