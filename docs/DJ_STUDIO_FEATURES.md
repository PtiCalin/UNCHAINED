# DJ Studio Features

## Multi-Deck Mixing
- Up to 6 decks (A–F)
- Dynamic add/remove decks
- Deck order and active deck persistence

## Hotkey Mapping
- F1–F6: Select active deck
- Space: Play/pause active deck
- J/K: Seek
- 1–8: Set/jump cues

## DeckManager UI
- Add/remove decks
- Visualize deck order
- Select active deck

## Cue/Loop/FX System
- Set/jump cues
- Set/clear loops
- Apply FX presets

## Analytics
- Track play history
- FX usage logging
- Recording management

## Integration
- Zustand state management
- SSE notifications
- Tauri desktop integration
