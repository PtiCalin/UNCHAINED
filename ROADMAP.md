# UNCHAINED Roadmap

## Phase 1 — Foundation (current)
- Repo setup (FastAPI + React/Tailwind + Tauri)
- Track import via API
- Metadata reading and SQLite persistence
- Basic library view

## Phase 2 — Analysis Engine
- Waveform generation
- BPM detection
- Musical key + Camelot
- Audio features (energy, valence, etc.)
- DB integration of analysis results

## Phase 3 — UI Modes
- Spotify-like Grid (HomeView)
- iTunes-like Table (TrackTableView)
- Vinyl Collector timeline (VinylCollectorView)
- Minimal Player (MinimalPlayerView)

## Phase 4 — Data Science
- Analytics view with histograms and embeddings
- Genre clusters and mood grids
- Artist network visualization

## Phase 5 — DJ Engine
- Deck A/B, beat grid, hot cues, loops
- Filters/EQ/effects, crossfader
- Sync engine (BPM align)
- Pad sampler

## Phase 6 — Mixing Suite
- Live recording
- Timeline editor
- Export mixes

## Phase 7 — Web Deployment
- FastAPI to cloud
- Storage to S3/Supabase
- SQLite → Postgres
- Authentication and sync
